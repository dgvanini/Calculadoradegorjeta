# Calculadoradegorjeta

E FAZ A DIVISÃO BASEADO NA QUANTIDADE DE PESSOAS.
EM JS ESSE FORAM USADAS VARIÁVEIS, getElementById, addEventListener, functions, innerHTML e windows.alert.
